# 伺服馬達模組教學

## 簡介
伺服馬達是一種可以精確控制角度的馬達，常用於機器人、遙控模型等項目。本教學使用標準的 SG90 伺服馬達。

## 硬體需求
- SG90 伺服馬達
- 連接線 x3
- 外部電源（選配，若使用多個伺服馬達）

## 接線說明
- 橙/黃線 -> PWM 腳位（訊號線）
- 紅線 -> 5V
- 棕/黑線 -> GND

## 功能特點
- 角度範圍：0-180度
- 高精度控制
- PWM 訊號控制
- 自帶回饋系統

## 程式範例
1. 基礎控制 (basic_servo.ino)
   - 角度控制
   - 速度控制
   - 位置校正

2. 掃描控制 (sweep_servo.ino)
   - 連續旋轉
   - 多段位置控制
   - 平滑運動

## 注意事項
- 不要超出角度限制
- 注意電源供應充足
- 避免機械卡死
- 初始化時的抖動屬正常現象
